# Tokenizing

## Operators and special characters

- plus: +
- minus: -
- multiply: *
- divide: /
- exponent: ^
- integer divide: \
- single-quote comment: '
- greater-than: >
- equals: =
- less-than: <

- less-than or equal: <=
- not equal: <>
- greater-than or equal: >=

- and: AND
- or: OR
- exclusive or: XOR
- equivalence: EQV
- implication: IMP
- modulo: MOD
- logical not: NOT

- print shorthand: ?

## DEF FNX and USR special cases

- Function definition: DEF FNNAME
    - Resulting function is named "FNNAME"
- USRX where X is a single digit 0 to 9

## Line numbers

- Line numbers are integers, can have preceeding 0
